Promptolian API

Compress AI prompts programmatically. Send text, get back shorter text and token metrics. No AI involved — all compression runs locally on the server using NLP rules. Privacy-first by design.

Overview

The Promptolian REST API exposes the same compression engine used in the CLI and transparent proxy. You send plain text; you get back compressed text plus token savings metrics.

REST API base URL: https://api.promptolian.com · Hosted proxy: https://proxy.promptolian.com

All requests and responses use JSON. The free tier (self-hosted) requires no authentication. Solo ($9/mo) and Team ($29/mo) plans include a hosted proxy and project API keys.

Solo plan ($9/mo) — hosted proxy, PostgreSQL sessions, 1 project API key, savings dashboard. Team plan ($29/mo) — up to 10 project keys plus per-project breakdown. Need more? Contact us.

Authentication

Pass your API key either in the JSON body or as a header. Free tier works without a key (standard tier only, IP rate-limited).

curl -X POST https://api.promptolian.com/compress \
  -H "Content-Type: application/json" \
  -d '{"text": "...", "tier": "pro", "api_key": "YOUR_API_KEY"}'
curl -X POST https://api.promptolian.com/compress \
  -H "Content-Type: application/json" \
  -H "X-API-Key: YOUR_API_KEY" \
  -d '{"text": "...", "tier": "pro"}'

Quickstart

Compress a prompt in one call — no API key needed for standard tier:

curl -X POST https://api.promptolian.com/compress \
  -H "Content-Type: application/json" \
  -d '{
    "text": "You are an expert Python developer. Please write a function to sort a list and return only the code without any explanation.",
    "tier": "standard"
  }'
import httpx

resp = httpx.post("https://api.promptolian.com/compress", json={
    "text": "You are an expert Python developer. Please write a function to sort a list and return only the code without any explanation.",
    "tier": "standard",
    "api_key": "YOUR_API_KEY",  # omit for free tier
})
data = resp.json()
print(data["compressed"])        # §EXP py developer. FN sort list →code ⊖explain.
print(data["tokens_saved_pct"])  # 21
const res = await fetch('https://api.promptolian.com/compress', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    text: 'You are an expert Python developer. Please write a function to sort a list and return only the code.',
    tier: 'standard',
    api_key: 'YOUR_API_KEY',
  }),
});
const data = await res.json();
console.log(data.compressed);
console.log(`Saved ${data.tokens_saved_pct}%`);

Response

{ "compressed": "§EXP py developer. FN sort list →code ⊖explain.", "original_tokens": 24, "compressed_tokens": 19, "tokens_saved": 5, "tokens_saved_pct": 21, "tier": "standard" }

POST /compress

Compress a single prompt. Returns the compressed text plus token metrics.

POST https://api.promptolian.com/compress

Request body

FieldTypeDescription
text*stringThe prompt text to compress. Max 50,000 characters.
tieroptstringstandard (default), pro, or developer. Higher tiers apply more aggressive NLP-based compression. Pro/developer require Solo or Team plan.
langoptstringLanguage hint: auto (default), en, es, fr, de, it.
api_keyoptstringYour API key. Alternatively pass as X-API-Key header. Required for pro/developer tiers.

Response fields

FieldTypeDescription
compressedstringThe compressed prompt text.
original_tokensintegerToken count of the input (BPE estimate).
compressed_tokensintegerToken count of the compressed output.
tokens_savedintegerTokens saved = original − compressed.
tokens_saved_pctintegerPercentage of tokens saved (0–100).
tierstringThe tier actually used.

POST /optimize-context

Prune and compress a multi-turn conversation history before sending it to an LLM. Removes off-topic messages, compresses retained ones, and optionally produces a rolling summary. Requires Solo or Team plan.

POST https://api.promptolian.com/optimize-context

Request body

FieldTypeDescription
messages*arrayArray of {"role": "user|assistant", "content": "..."} objects.
query*stringThe current user query. Used to score relevance of past messages.
summaryoptstringRolling summary from a previous call. Injected as context.
modeoptstringlossless (default) — removes only clearly off-topic messages. aggressive — prunes more heavily.
api_keyoptstringSolo or Team plan key required.
 
curl -X POST https://api.promptolian.com/optimize-context \
  -H "Content-Type: application/json" \
  -d '{
    "messages": [
      {"role": "user",      "content": "I am building a FastAPI app"},
      {"role": "assistant", "content": "Great, I can help with FastAPI."},
      {"role": "user",      "content": "What is the weather today?"},
      {"role": "assistant", "content": "I am not sure about the weather."},
      {"role": "user",      "content": "Back to FastAPI — how do I add auth?"}
    ],
    "query":   "How do I add JWT authentication to FastAPI?",
    "mode":    "lossless",
    "api_key": "YOUR_API_KEY"
  }'
resp = httpx.post("https://api.promptolian.com/optimize-context", json={
    "messages": [
        {"role": "user",      "content": "I am building a FastAPI app"},
        {"role": "assistant", "content": "Great, I can help with FastAPI."},
        {"role": "user",      "content": "What is the weather today?"},
        {"role": "user",      "content": "Back to FastAPI — how do I add auth?"},
    ],
    "query":   "How do I add JWT authentication to FastAPI?",
    "mode":    "lossless",
    "api_key": "YOUR_API_KEY",
})
data = resp.json()
print(data["optimized_prompt"])
print(f"{data['messages_pruned']} messages pruned, {data['tokens_saved_estimate']} tokens saved")

Response

{ "optimized_prompt": "[compressed conversation ready to send to LLM]", "original_tokens": 87, "optimized_tokens": 54, "tokens_saved_estimate": 33, "messages_pruned": 2, "new_summary": "User building FastAPI app, asked about auth." }

POST /compress-tools

Compress a JSON tool schema array (OpenAI / Anthropic format) into a compact DSL. Achieves ~69% compression on first call, ~90% session average via session caching. Requires Solo or Team plan.

POST https://api.promptolian.com/compress-tools
FieldTypeDescription
tools*arrayArray of tool objects in OpenAI or Anthropic format.
session_idoptstringUnique ID for this conversation session. Enables cache-hit compression (~90% session avg CR) on repeated calls.
api_keyoptstringSolo or Team plan key required.
 
curl -X POST https://api.promptolian.com/compress-tools \
  -H "Content-Type: application/json" \
  -d '{
    "tools": [
      {
        "name": "get_weather",
        "description": "Get the current weather for a location",
        "input_schema": {
          "type": "object",
          "properties": {
            "location": {"type": "string", "description": "City name"},
            "unit":     {"type": "string", "enum": ["celsius","fahrenheit"]}
          },
          "required": ["location"]
        }
      }
    ],
    "session_id": "my-chat-session-123",
    "api_key":    "YOUR_API_KEY"
  }'
tools = [{
    "name": "get_weather",
    "description": "Get the current weather for a location",
    "input_schema": {
        "type": "object",
        "properties": {
            "location": {"type": "string", "description": "City name"},
            "unit":     {"type": "string", "enum": ["celsius", "fahrenheit"]},
        },
        "required": ["location"],
    },
}]

resp = httpx.post("https://api.promptolian.com/compress-tools", json={
    "tools":      tools,
    "session_id": "my-chat-session-123",
    "api_key":    "YOUR_API_KEY",
})
data = resp.json()
print(data["dsl"])             # compact DSL string
print(data["tokens_saved"])   # tokens saved vs raw JSON

Compression tiers

TierWhat it doesAvg CRPlan required
standardSymbol substitution + grammar rules. Removes filler phrases, replaces verbose patterns with symbols (§EXP, →code, ∑, etc).~9%Free / Solo / Team
proStandard + math operators, numeric abbreviations (5 million → 5M), verbose phrase removal.~10%Solo / Team
developerPro + spaCy NLP deep pruning, domain-specific packs (code, finance, legal, medical), dependency parse.~11%, up to 33%Solo / Team
Token counts use BPE estimation via tiktoken (cl100k_base encoding — same as Claude and GPT-4). Both original and compressed counts are measured so you can verify savings independently.

CLI

The promptolian CLI runs compression locally — no server needed for standard tier.

Install

pip install promptolian

# With NLP engine (pro/developer tiers):
pip install "promptolian[nlp]"
python -m spacy download en_core_web_sm

Usage

# Compress a prompt (standard tier, no server needed)
promptolian compress "You are an expert Python developer. Return only the code."

# Pro tier with JSON output
promptolian compress "You are an expert Python developer." --tier pro --json

# Compress from file
promptolian compress --file prompt.txt --tier developer --output compressed.txt

# Optimize conversation history
promptolian context --file chat.json --query "How do I add auth?"

# Show compression statistics
promptolian stats

Self-hosted

Run the Promptolian API on your own machine or infrastructure. Prompts are processed locally — nothing is sent to api.promptolian.com.

Option A — Docker (recommended)

One command. SQLite data persisted in a Docker volume. No account needed.

# Clone the public repo
git clone https://github.com/Maurizio-L/promptolian-public
cd promptolian-public

# Start (builds image on first run, ~2 min)
docker compose up

# Test it
curl http://localhost:3001/health
curl -X POST http://localhost:3001/compress \
  -H "Content-Type: application/json" \
  -d '{"text": "Please explain step by step", "tier": "pro"}'

Then in your code, change one line: API_BASE = "http://localhost:3001" instead of https://api.promptolian.com. All endpoints work identically.

Option B — Python (from repo)

git clone https://github.com/Maurizio-L/promptolian-public
cd promptolian-public
pip install -r public/requirements-selfhost.txt
python -m spacy download en_core_web_sm

# Start on default port 3001
python -m promptolian.server

# Or on a custom port
python -m promptolian.server --port 8080

Option C — pip install

pip install "promptolian[self-hosted]"
python -m spacy download en_core_web_sm
promptolian-server              # starts on localhost:3001
promptolian-server --port 8080  # custom port

Privacy guarantee
When self-hosted, every compression happens inside your own process. No data is sent to api.promptolian.com or any external service. The SQLite database (or your PostgreSQL instance) stays on your infrastructure.
