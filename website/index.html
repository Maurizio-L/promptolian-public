Get started in 3 steps
Pick the option that fits your setup.
Python developer
Patch your client
One function call. Every Anthropic or OpenAI request is compressed automatically from that point on.
pip install promptolian

from promptolian import patch_anthropic
patch_anthropic()
# done — all calls compressed
Any language · recommended
Run the proxy
A local proxy intercepts every call and handles tool schema caching automatically. Works with any HTTP client.
pip install "promptolian[proxy]"
promptolian proxy

# change one line in your code:
client = anthropic.Anthropic(
  base_url="http://localhost:3002"
)
Claude Code user
MCP install
Adds Promptolian as a native tool inside Claude Code. Compress prompts and schemas without leaving your editor.
pip install "promptolian[mcp]"
promptolian mcp install
# restart Claude Code — done
All options are free to start. No account required. Sessions and stats are stored locally on your machine.
